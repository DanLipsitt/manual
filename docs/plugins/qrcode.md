#QSQRCode

Displays objects as QR Code.

 Summary                  | &nbsp; 
-------------------------:|:--------------------
 Latest plugin version    | 1.0.0
 Supported MacOS versions | 10.11, 10.12, 10.13
 Quicksilver builds       | 4024


### Description

Displays objects as [QR Code](http://en.wikipedia.org/wiki/QR_Code) 2D
barcodes
