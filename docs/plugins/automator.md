## Automator Plugin

The Automator plugin adds a single 'Execute Workflow' action for running
Automator workflows.

### Actions

**Execute Workflow**

With an Automator workflow selected in Quicksilver's first pane, the 'Execute
Workflow' action runs the workflow.
